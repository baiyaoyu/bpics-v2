## 关于这个项目

bpics-v2 正如标题所展示的，这仅仅是一个个人网站项目（Just For Fun），目前主要实现的功能是实现文件共享，方便异地访问、分享有趣的视频，音乐等。

整个项目使用了go语言开发，使用了gin这一流行的web框架，目前使用了go的模板语法。

后续计划：

+ 添加认证功能
+ 添加博客功能
